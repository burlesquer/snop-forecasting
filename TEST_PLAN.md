## Affected Pages/Routes

- `/` (Executive Dashboard) — KPI 카드, 위험/과잉 테이블, 카테고리 차트, 시뮬레이터
- `/analyst` (Analyst View) — 모델 비교, SKU 시계열, SHAP plots
- `/_health` (Vercel auto, no test needed)

## Key Interactions to Verify

- 신뢰구간 토글 on/off → fan band fade transition (250ms)
- What-if 토글 on/off → P50 라인 path morph (300ms)
- 시뮬레이터 SKU 선택 + 발주량 입력 → gauge + 4주 추이 즉시 갱신
- Executive → Analyst 탭 전환 (300ms 내)
- SKU 드롭다운 → 시계열 차트 + SHAP force plot 동시 갱신

## Edge Cases

- 발주량 0 입력 → 현재 추세대로 시뮬레이션
- 발주량 음수 입력 → input border danger + helper text
- SKU 미선택 → simulator empty state + dropdown 강조
- promotion 이력 0인 SKU → What-if 토글 disabled + 툴팁
- 빈 risk SKU 리스트 → "🌿 위험 SKU 없음" 긍정 empty
- JSON fetch 실패 → "데이터 로드 실패" 메시지 + 새로고침 안내
- SHAP top-5 외 SKU 선택 → "전체 분포 모드" fallback
- 모바일 320px → simulator disabled + 안내 카드
- 빠른 토글 클릭 (debounce) → transition cancel/queue

## Critical Paths (E2E 필수)

1. **데모 5분 시연 흐름**: 랜딩 → KPI → 위험 테이블 → 신뢰구간 토글 → What-if 토글 → 시뮬레이터 사용 → Analyst 탭 → SHAP 클릭 → 종료
2. **시뮬레이터 의사결정 루프**: SKU 선택 → 발주량 입력 → 게이지 갱신 → 다른 발주량 → 비교
3. **반응형 렌더링**: 1440 → 1024 → 768 → 320 모든 breakpoint에서 레이아웃 깨지지 않음
